## App URL

The app is deployed at:- https://feed-app-five.vercel.app/

To run the development server please use:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3002](http://localhost:3002) with your browser to see the result.
